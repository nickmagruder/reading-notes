# JQuery
* Find elements using CSS-style selectors
* Do something with said elements using jQuery methods
* $() is the shortcut
* Estimated to be used on 25% of websites because of its simplification
* Simple Selectors, Common Tasks in Less Code, Cross-browser compatibility

## Matched Set/Jquery selection
* when one or multiple elements are selected, a jQuery object is returned
* a matched set or jQuery selection
* query methods a cam get and set data
* jquery objects are array-like, and store elements in the same order as they are in the HTML
* repeatedly used jquery selections should be stored in variables: $jQueryVariable
